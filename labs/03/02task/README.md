# Задание 2

## В чём различия между видами конструкторов?

Конструктор по умолчанию вызывается без аргументов и инициализирует объект значением по умолчанию. 

Конструктор из std::string принимает строку и копирует её в поле. Конструктор из const chadr* принимает строковый литерал и тоже сохраняет его. Копирующий конструктор принимает другой объкт того же класса и делает его копию

## Когда вызывается копирующий конструктор?

При инициализации через оператор присвоения TrackedString d = c

При передаче объекта в функцию по значению

Нужен для того чтобы правильно скопировать объект. Без него компилятор генерирует побитовую копию

## Выводы по лог-файлу

Объекты создаются в том порядке, в котором написаны в коде. Уничтожаются в обратном порядке LIFO. При `d = c` вызывается копирующий конструктор потому что это иницализация, а не присваивание
